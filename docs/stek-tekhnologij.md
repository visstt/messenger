# Стек технологий мессенджера


## Инфраструктура

- **Docker Compose** — запуск всех сервисов одной командой.
- **PostgreSQL** — хранение пользователей, чатов, сообщений, настроек.
- **Nginx** — раздача собранного веб-приложения 
- **S3-совместимое хранилище** — файлы и вложения (плюс локальный диск в контейнере).
- **LiveKit** — аудио- и видеозвонки в чатах.


---

## Backend (серверная часть)

- Язык **Go**.
- REST API для клиентов: регистрация, вход, чаты, сообщения, профили, загрузка файлов.
- **WebSocket** — доставка новых сообщений и статусов «в сети» без перезагрузки страницы.
- Авторизация через **JWT** в cookie (отдельная схема для админов).


---

## Frontend (веб-клиент)

- **React** и сборщик **Vite**.
- PWA (можно установить как приложение в браузере).
- Звонки через клиент LiveKit в браузере.

---

## Desktop

- **Electron** — окно браузера с тем же веб-мессенджером (иконка в трее, уведомления). Отдельного «нативного» протокола нет.

---


## Сводка

| Часть | Технологии |
|-------|------------|
| Данные | PostgreSQL |
| API и realtime | Go, REST, WebSocket |
| Интерфейс | React, Vite, Nginx |
| Звонки | LiveKit |
| Файлы | S3 + локальное хранилище |
| Десктоп | Electron → тот же сайт |
